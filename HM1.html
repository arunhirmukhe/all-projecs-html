1.1 Introduction to the front end
1.1.1 What is the front-end?

The front end concerning the website is the view of the web page that the user sees on the screen or the interface on the screen with which he can interact. The front end is very important for the overall look and feel of the website. Front-end must be developed by keeping users’ perspectives in mind so that the design will be user-friendly. The front end plays an important role in the success of any website. Hence the front end needs to be designed properly.


1.1.2 Design considerations for the front-end

following considerations must be taken into account for designing a good front-end

  • Grid layout system
  • Color combination
  • Graphics and Animations
  • Fonts and Texts
  • Links and navigation for additional information
  • Relevant mages/Icons
  • Proper Forms and buttons
  • Responsive elements
  • Content Organization

1.1.3 Technologies used to design front end

Below mentioned are some of the technologies used to design front end

1.1.3.1 Basic Technologies
  • HTML (HyperText Markup Language): This is a standard and basic scripting language used to design web pages.
  • CSS (Cascaded Style Sheet): CSS is used to design the feel and appearance of HTML elements on the screen. CSS help to reduce the HTML code size and can optimize the coding.
  • JavaScript: JavaScript is used to design dynamic content of websites. This is a lightweight and most commonly used scripting language in web development.

1.1.3.2 Advanced(Modern) Technologies
  • React: React is a more popular technology in recent times in front-end development. This is an open-source library developed and released by MIT in the year 2013. React is maintained and supported by Facebook.
  • Angular: Angular JavaScript is a front-end web development tool designed in 2009 and licensed by MIT as an open-source framework. AngularJS is maintained by Google.
  • Vue.js: This front-end technology was released in 2014 and its highly appreciated by the programmer’s community and may lead to front-end development in near future.
